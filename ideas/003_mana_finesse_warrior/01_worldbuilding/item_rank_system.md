# 世界観設定: アイテム等級（ランク）制度

## 舞台の概要
- この世界の装備・道具は「武器だから強い」ではなく、**魔力をどれだけ“綺麗に通せるか”**で性能が決まる。
- ただし流通・契約・大会運営の都合で、市場では「強さ」を**単一のランク**に落とし込んで扱う文化がある。
- ランクは“魔力の性質”だけで決まらず、**攻撃力／防御力／汎用性（付加機能）／扱いやすさ**などの総合で決まる（用途が違っても比較できるよう、最低限の共通物差しにする）。

## 地理・場所
- **王国鑑定協会（きょうかい）**
  - 都市ごとに支部を持つ半官半民組織。等級の基準を定め、鑑定票の偽造を取り締まる（が、賄賂で穴もある）。
- **刻印工房街（こくいんこうぼうがい）**
  - 魔導具職人と刻印師が集まる区画。表通りは公認品、裏通りは“検品だけ通す”改造や、禁具の部品売買が行われる。
- **整備屋（せいびや）**
  - 武器や防具の癖・反り・刻印の“噛み合わせ”を直す職。貴族は専属を抱え、平民は大金を貯めて一度頼めるかどうか。

## 魔法・技術体系
- **アイテムランク**
  - 攻防・機能・扱いやすさをまとめて扱うための等級制度。鑑定協会が基準を定め、鑑定票にランクを記載する。

### 1) ランクの決まり方（鑑定協会の基準）
- 表向きは「ランク」のみが流通するが、協会内部では以下を点検し、総合点でランクを決める。
  - **攻撃力**: 破壊・貫通・切断などの実効（対魔力壁／対鎧を含む）。
  - **防御力**: 受け流し／耐久／自己修復／結界付与などの実効。
  - **汎用性**: 付加機能の数と質（例：属性切替、視界補助、封印、対毒、通信）。
  - **扱いやすさ**: 反動、暴発のしやすさ、必要技能、維持費（触媒・充填）、整備難度。
  - **信頼性**: 劣化速度、故障時の危険性（逆流・焼損・破片化など）。

### 2) ランク表（市場で通じる単一表記）
- 市場では `E < D < C < B < A < S < SS < SSS` が一般的。
  - **E**: 粗悪〜学習用。戦闘は非推奨（壊れ方が危ない）。
  - **D**: 量産品。民兵・下級冒険者の主力。
  - **C**: 実戦級。冒険者の標準ライン。
  - **B**: 上物。機能か性能の“尖り”が出始める。
  - **A**: 高級品。強い上に扱いやすい（ここから貴族が抱え込みやすい）。
  - **S**: 希少。軍用・貴族用の切り札級が混じる。
  - **SS**: 国家級。単品で戦局や都市防衛のバランスが変わる。
  - **SSS**: 伝説級。遺物・禁具に近く、売買そのものが事件になる。

### 3) ランク運用の“罠”（単一ランクの限界）
- 協会ランクはあくまで流通のための“総合値”。同ランクでも方向性は違う。
  - **攻撃特化のA**: 強いが脆い／整備費が重い、などの欠点が出やすい。
  - **汎用性寄りのA**: 便利だが瞬間火力はB相当、などのズレが起きる。
- そこで現場（冒険者ギルド、軍、護衛会社）は、ランクに加えて「用途札（攻／防／補／特）」を付けて誤解を減らす。

### 4) 禁具（きんぐ）と規制
- 危険性が高い魔導具は、ランクに関わらず **禁具指定** を受ける。
  - 例: 逆流で相手の魔力回路を裂く、魂や記憶を“縫い止める”、結界監視を無効化する等。
- 王国は「総合A以上の攻性武具」「禁具指定」の所持に登録を求め、無登録は重罪（ただし貴族は抜け道がある）。

## 歴史・現状
- かつては貴族が等級を独占し、兵器管理の名目で市井の高位品を没収していた（身分制度の補強）。
- 近年、鑑定協会が“公称ランク”を整備し、冒険者・商人でも売買可能になったが、裏市場は鑑定票の偽造や改竄で肥大化している。
- 戦後の軍縮や横流しで人体強化向けの補助具（籠手・帯・針など）が増え、「戦士が道具で伸びる」流れが加速した。

## 文化・風習
- 武闘大会や決闘には「持ち込み上限（総合Bまで等）」があるが、**攻撃力だけ尖っていて扱いにくい武器**など“抜け道”が人気。
- 高位品ほど「持ち主を選ぶ」という迷信があり、相性が悪いと怪我をする＝道具に嫌われた、と語られる。
- 一流職人は名を刻まない。代わりに内部構造に“職人癖”という署名が残り、鑑定士だけがそれを読む。

## おもしろい設定の羅列（使いやすいパーツ集）
- 高ランク品ほど「強い」より「欠点が少ない（扱いやすい／壊れにくい）」が値段に効く。
- 裏市場の偽造は、攻撃力だけ盛る・検品だけ通す改造が主流。実戦で“壊れ方が派手”でバレる。
- レンは「武器のランク」より「付加機能の噛み合わせ」で勝つ（剣そのものが最上位である必要はない）。
- ミリアは高ランクの媒体を持っていても、運用が下手だと実効は落ちる（道具は持ち主の差を拡大する）。
